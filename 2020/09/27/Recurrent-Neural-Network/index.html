Recurrent Neural Network

Notes on Recurrent Neural Network

RNNs is proposed to solve the problems with sequential information. RNNs are called recurrent because they perform the same task for every element of a sequence, with the output being depended on the previous computations.

Traindata: $\{X_i\}_{i=1}^N.$ For each $X_i$: $X_i=[X_i^1,\dots,X_i^m]$ (m sequence length).

In the training loop $t$, we select all the $\{X_i^t\}_{i=1}^N$ from the batch samples as the input data.
