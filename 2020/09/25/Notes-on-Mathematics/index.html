Some Notes on Mathematics

1. Cardinality

The cardinality of a set is a measure of a set’s size, meaning the number of elements in the set. When $A$ is finite, $|A|$ is simply the number of elements in $A$. When $A$ is infinite, $|A|$ is represented by a cardinal number

For instance, the set $A = \{1,2,4\}$ has a cardinality of 3.
