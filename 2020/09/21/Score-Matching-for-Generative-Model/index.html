Score Matching for Generative Model

Generative Modeling by Estimating Gradients of the Data Distrbution - NIPS 2019

(1) Main Contribution

  • Introduce a new generative model, where samples are produced via Langevin dynamics using gradients of $p(x;\theta)$ with score matching.
  • Since score gradients are hard to estimate in low-dimensional manifolds, the authors perturb the data with gaussian noise and instead estimate the score function of the perturbed data distribution.
  • Annealed Langevin dynamics methods can gradully decrease the noise level as the sampling process gets close to the data manifold.
  • Their methods avoid sampling process during the training.
  • The score function is parameterized with a neural network instead of kernel functions.

(2) Background & Motivation

  • Generative models have many applications in machine learning. Basically, we want to train a generative model $p(x;\theta)$ to simulate the sample generating process of $p_{data}(x)$. Recent works are mainly driven by two approaches: likelihood-based methods (e.g. VAE) and generative adversarial networks.
  • Likelihood-based models normally maximize the log-likelihood or the surrogate evidence lower bound, but the difficulty of that has been attributed to the partition function (or named normalization constant). The normalized probability density has to be non-negative and integrated to 1, so the energy-based models or some specialized architectures (e.g. flow models) are widely used in this scenia.
  • Generative adversarial networks have achieved great success in recent few years, but the training can be unstable due to the adversarial objective and often criticized with the mode collapse problem.
  • Noise contrastive estimation and minimum probability flow typically only work well in low-dimensional cases.

(3) Proposed methods

Notation: Define the score of a probability density $p(\mathrm{x})$ as $\nabla_{\mathrm{x}}\log p(\mathrm{x})$. The score network $S_{\theta}:\mathbb{R}^D\rightarrow \mathbb{R}^D$ is a neural network parameterized by $\theta$.

Goal: Use the dataset to learn a model for generating new samples from $p_{data}(\mathrm{x})$.

(3.1) Score matching for score estimation

$$
\begin{align}
&\text{minimize } \frac{1}{2}\mathbb{E}{p{data}}[|S_{\theta}(\mathrm{x})-\nabla_{\mathrm{x}}\log p_{data}(\mathrm{x})|^2_2] \
\Leftrightarrow &\text{minimize } \mathbb{E}{p{data}}[\text{tr}(\nabla_{\mathrm{x}}{S_{\theta}(\mathrm{x})})+ \frac{1}{2}|S_{\theta}(\mathrm{x})|^2_2]
\end{align}
$$

Where $\nabla_{\mathrm{x}}S_{\theta}(\mathrm{x})$ denotes the Jacobian of $S_{\theta}(\mathrm{x})$, which is not scalable to high dimensional data.

Denosing score matching can completely circumvents $\text{tr}(\nabla_{\mathrm{x}}{S_{\theta}(\mathrm{x})})$. It perturbs the data point $\mathrm{x}$ with a pre-specified noise distribution $q_{\delta}(\tilde{\mathrm{x}}|\mathrm{x})$ and instead employing score matching to estimate the perturbed data distribution $q_{\delta}(\tilde{\mathrm{x}})\triangleq \int q_{\delta}(\tilde{\mathrm{x}}|\mathrm{x})p_{data}(\mathrm{x}) d\mathrm{x}$. Then the objective function is:
$$
\begin{align}
\frac{1}{2} \mathbb{E}{q{\delta}(\tilde{\mathrm{x}}|\mathrm{x})p_{data}}[|S_{\theta}(\tilde{\mathrm{x}})-\nabla_{\tilde{\mathrm{x}}}\log q_{\delta}(\tilde{\mathrm{x}}|\mathrm{x})|^2_2]
\end{align}
$$
Sliced score matching uses random projections to approximate the trace term in (2). The objective is
$$
\begin{align}
\mathbb{E}{p{\mathrm{v}}}\mathbb{E}{p{data}}[\mathrm{v}^T \nabla_{\mathrm{x}}{S_{\theta}(\mathrm{x})}\mathrm{v}+ \frac{1}{2}|S_{\theta}(\mathrm{x})|^2_2]
\end{align}
$$
Where $p_{\mathrm{v}}$ is a simle distribution of random vectors.

(3.2) Sampling with Langevin dynamics

Given a fixed step size $\epsilon>0$ and an initial value $\tilde{\mathrm{x}}0\sim \pi({\mathrm{x}})$ with $\pi$ is a prior distribution, $\mathrm{z}_t \sim \mathcal{N}(0,I)$,
$$
\begin{align}
\tilde{\mathrm{x}}_t = \tilde{\mathrm{x}}{t-1} + \frac{\epsilon}{2} \nabla_{\mathrm{x}} \log p(\tilde{\mathrm{x}}_{t-1}) + \sqrt{\epsilon}\mathrm{z}_t
\end{align}
$$
